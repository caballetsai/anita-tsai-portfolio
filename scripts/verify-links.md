# Link Verification Workflow

Before moving an item from `src/content/draft-works.json` into `src/content/works.json`, verify:

1. The cover image exists in `public/assets/covers/`.
2. The Chinese title matches the publication page or source material.
3. The author line includes 蔡靚萱 / Anita Tsai / Ching-Hsuan Tsai.
4. The external URL opens without needing private credentials, or the card clearly notes login requirements.
5. The source is one of: Business Weekly archive/article page, Wealth Magazine, United Daily News, LinkedIn Featured, or another user-approved public work page.
6. The image source note is clear enough to trace back to PPT or a public page.
